# Conversational Multilingual Language Learning Assistant

This project is a Python-based conversational language learning assistant designed to help users learn and practice multiple languages through real-time interaction. The system automatically detects the language of user input, translates text between different languages, and provides meaningful responses to improve understanding and communication skills.

## Features
- Automatic language detection
- Multilingual translation support
- Interactive conversation with users
- Simple and user-friendly GUI
- Helps improve language skills

## Technologies Used
- Python
- Tkinter (for GUI)
- Language detection library
- Translation API

## How to Run
1. Install required libraries.
2. Run the file: Languageassistant.py
3. Start interacting with the assistant.
